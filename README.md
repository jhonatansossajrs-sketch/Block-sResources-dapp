# Block's Resources – Trazabilidad de Recursos Ambientales en Stellar + Soroban

Smart contract multisig (2 de 2) para rastrear recursos ambientales (agua, madera, minerales sostenibles, etc.) en la red Stellar Testnet.

### Características
- Multisig nativo (2 firmas requeridas)
- Trazabilidad inmutable con metadatos ambientales
- Liberación automática de fondos al receptor verificado
- Frontend Next.js + Freighter Wallet
- Deployado y probado en Testnet
